How to install GS & LS database:

1) Open Install.bat in notepad

2) Find: 
set user=root
set pass=
set LSDBname=au_server_ls
set GSDBname=au_server_gs
set AEDBname=aion_emu
set DBHost=localhost
/\ /\ /\ /\ /\ /\ /\ /\ /\ /\ /\
and change it to your properties

3) Save changes

4) Run Install.bat
